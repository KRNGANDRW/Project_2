Циклы в C++

Иногда необходимо повторять одно и то же действие несколько раз подряд. Для этого в программировании используют циклы.

Цикл for

Если известно, точное количество действий (итераций) цикла то можно использовать цикл for.
Синтаксис:

for (начальное значение счётчика; конечное значение счётчика; шаг)
{
Тело цикла (дейсьвия, выполняемые в цикле)
}

Счётчик цикла — значение, в котором хранится количество проходов цикла.

Пример кода

Напишем программу, которая будет считать сумму чисел от 1 до 100.

#include <iostream>
using namespace std;
int main()
{
int i;
int sum = 0;
for (i = 1; i <= 100; i++)
{
sum = sum + i;
}
cout << "Сумма чисел от 1 до 100 = " << sum << endl;
return 0;
}

Цикл while

Когда мы не знаем, сколько итераций должен произвести цикл, нам понадобится цикл while или do...while. Синтаксис цикла while в C++ выглядит следующим образом:

while (начальное значение счётчика; конечное значение счётчика; шаг)
{
Тело цикла
}

Данный цикл будет выполняться, пока условие, указанное в круглых скобках является истиной.

Реализуем решение выше поставленной задачи, только через цикл while.

#include <iostream>
using namespace std;
int main()
{
int i = 0;
int sum = 0;
while (i < 100)
{
i++;
sum += i;
}
cout << "Сумма чисел от 1 до 100 = " << sum << endl;
return 0;
}

Цикл do while

Цикл do while имеет много общего с циклом while. Отличие заключается в том, что в цикле do while условие выполняется как минимум один раз.

#include <iostream>
using namespace std;
int main()
{
int i = 0;
int sum = 0;
do
{
i++;
sum += i;
}
while (i < 100);
cout << "Сумма чисел от 1 до 100 = " << sum << endl;
return 0;
}
